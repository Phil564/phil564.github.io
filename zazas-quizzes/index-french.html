Zaza's Quizzes

Un court jeu de Roblox qui se joue à 4 joueurs avec pass play, que j'ai fait pour mon cours de Science avec 3 de mes amis.

Téléchargez maintenant!

Un court jeu de Roblox que j'ai fait pour mon cours de Science. Puisque ma langue primère est le français, et que mon école est une école francophone, j'ai fait le jeu uniquement en français. Par contre, une version anglaise pourrait sortir un jour.

Le nom original du jeu était "Atomic Classroom". Je voulais originalement que le nom du jeu soit "Molecular Madness", mais ils n'aimaient pas le nom. Donc plus tard dans la semaine, je leur ai suggéré de changer le nom du jeu à "Zaza's Quizzes", et ils ont trouvé le nom meilleur que "Atomic Classroom" (et ils m'ont ensuite répondu avec "C'est qui le con qui a décidé que le nom du jeu saura Atomic Classroom???").

Un jour, je vais probablement refaire le jeu dans Unity, car vous avez besoin d'une version spéciale de Roblox Studio pour jouer au jeu en ce moment.

Comment jouer?

  • Premièrement, vous aurez besoin de la version "FiB v16" de Roblox Studio. Vous pouvez l'obtenir ici. (pour Windows et Mac seulement)
  • Téléchargez le jeu.
  • Placez le fichier "content-zaza_s_quizzes" de l'archive que vous venez de téléchargez dans le fichier "content" de Roblox Studio.
  • Démarrez l'executable de Roblox Studio hors-ligne (ou en mode avion). Vous avez besoin de le démarrer hors-ligne, car l'executable va essayer de se connecter aux serveurs de Roblox (et les serveurs ne supportent plus cet executable). Lorsque vous le démarrez hors-ligne, vous allez "skipper" le menu de régistration
  • Ouvrez le fichier Roblox qui est inclue dans l'archive avec l'executable (vous devez le "dragger" dans l'executable, ouvert)
  • Bravo! Vous pouvez maintenant jouer au jeu! (oh et vous aurez besoin de 3 autres joueurs pour y jouer.)
